    Manuales de usuario

    1. Instrucciones básicas

    Estas son las instrucciones básicas para instalar, configurar y usar el software LUMINO.

    Acceso a la aplicación

    1. Accede a la página:
      • Abre tu navegador web e ingresa la dirección de LUMINO:
        • Si está desplegada en un servidor, usa la URL proporcionada (por ejemplo, http://lumino.com).
        • Si estás ejecutando la aplicación en local, ingresa http://127.0.0.1:8000.
    2. Página de bienvenida:
      • Verás una pantalla de bienvenida con una imagen divertida y opciones para Iniciar sesión o Registrarte.
      • En la barra lateral, encontrarás la opción de cambiar el idioma (español o inglés) y el modo (oscuro o claro).

    Registro e inicio de sesión

    1. Registrarte:
      • Haz clic en Registrarte.
      • Rellena el formulario con tus datos: nombre de usuario, contraseña, nombre, apellidos y correo electrónico.
      • Haz clic en Registrarse.
      • ¡Listo! Ya tienes una cuenta en LUMINO.
    2. Iniciar sesión:
      • Si ya tienes una cuenta, haz clic en Iniciar sesión.
      • Introduce tu nombre de usuario y contraseña.
      • Serás redirigido a la página principal (/subjects/).

    Página principal (Subjects)

    • Al iniciar sesión, serás redirigido a la página principal, donde verás las asignaturas en las que estás matriculado (si eres estudiante) o las asignaturas que impartes (si eres profesor).
    • Si eres estudiante, también verás la opción de Pedir un certificado (solo disponible si tienes al menos una asignatura con nota).
      • Al hacer clic, se enviará automáticamente un PDF con tu certificado al correo electrónico registrado.
      • Si no tienes notas, aparecerá un mensaje de error.

    2. Guías avanzadas

    Aquí se describen funcionalidades específicas y tareas complejas para usuarios avanzados.

    Funcionalidades para estudiantes

    1. Matricularse en asignaturas:
      • En la barra lateral, haz clic en Inscribirte.
      • Selecciona las asignaturas disponibles marcando los checkboxes.
      • Haz clic en Matricularse.
      • Las asignaturas seleccionadas aparecerán en tu página principal.
    2. Ver lecciones y notas:
      • En la página principal, selecciona una asignatura para ver sus lecciones.
      • Si tienes una nota en la asignatura, esta se mostrará en la parte superior.
      • Haz clic en una lección para ver su contenido.
    3. Editar perfil:
      • Haz clic en tu avatar en la barra de navegación.
      • Selecciona Perfil para ver y editar tu información.
      • Puedes añadir una imagen de avatar y una biografía.
      • Guarda los cambios para actualizar tu perfil.

    Funcionalidades para profesores

    1. Gestionar asignaturas:
      • En la página principal, selecciona una asignatura que impartas.
      • Verás opciones para Añadir lección, Editar lección o Eliminar lección.
      • Haz clic en Añadir lección para crear una nueva lección con un título y contenido.
    2. Calificar estudiantes:
      • En la página de una asignatura, haz clic en Poner notas.
      • Verás una lista de estudiantes matriculados.
      • Introduce la nota correspondiente para cada estudiante y guarda los cambios.

    3. Solución de problemas

    Aquí se responden preguntas frecuentes y se proporcionan soluciones a errores comunes.

    Preguntas frecuentes

    1. ¿Cómo cambio mi contraseña?:
      • Actualmente, LUMINO no tiene una opción para cambiar la contraseña desde la interfaz. Contacta con el administrador del sistema para solicitar un cambio.
    2. ¿Qué hago si no puedo iniciar sesión?:
      • Verifica que estás utilizando el nombre de usuario y la contraseña correctos.
      • Si olvidaste tu contraseña, contacta con el administrador del sistema.
    3. ¿Cómo puedo desmatricularme de una asignatura?:
      • Ve a la página principal y selecciona Desmatricularse en la barra lateral.
      • Marca las asignaturas de las que deseas desmatricularte y haz clic en Confirmar.

    Errores comunes

    1. Error al pedir un certificado:
      • Si no tienes notas en ninguna asignatura, no podrás solicitar un certificado. Asegúrate de que tus profesores han calificado tus asignaturas.
    2. Error 404: Página no encontrada:
      • Verifica que la URL es correcta. Si el problema persiste, contacta con el administrador del sistema.
    3. Error al añadir una lección (profesores):
      • Asegúrate de que estás conectado como profesor y que tienes permisos para gestionar la asignatura.
      • Si el problema persiste, contacta con el administrador del sistema.

    Note

    Consejo: Asegúrate de seguir los pasos de inicio de sesión correctamente para evitar problemas con el acceso.

    Warning

    Advertencia: Si olvidas tu contraseña, asegúrate de contactar al administrador para solucionarlo.

    Tip

    Sugerencia: Si no ves las asignaturas en tu página principal, verifica que estés matriculado correctamente.

    Danger

    Error crítico: Si la página principal no carga correctamente, es posible que haya un problema con la conexión o con el servidor. Contacta al administrador del sistema.
